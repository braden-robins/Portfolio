It was a bit confusing at first but near the end things started to make sense. This was our first website's html.

Trial Website CSS:

Now it was time to add the CSS. We learnt how to add classes (names essentially) to each section in our html. This allowed us to add CSS to individual aspects of our code. We learnt how to align items, change font weight, size, and style, how to add colour, and how to move things around with margin and padding. What you see on the right is the result of that.
